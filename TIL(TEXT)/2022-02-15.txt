<HTML+CSS+Javascript 웹 프로그래밍>

CHAPTER 1.
- What is web?
>>> world wide web:세상의 전체 컴퓨터를 연결하는 거미줄
   거미줄처럼 연결된 인터넷에서 정보를 서로 교환한다는 뜻.
   인터넷을 이용한 정보 교류 시스템.

- 웹과 인터넷의 다른 점
>>> 인터넷이 고속도로라면 웹은 고속도로를 이용한 서비스 상품
인터넷은 통신망, 웹은 이 통신망 위에서 이루어지는 서비스 중 하나.

- 웹을 세상에 내놓은 사람
>>> 팀 버너스리가 연구원들끼리 인터넷 상에서의 문서공유 시스템으로
    만든 것이다.

- 일반인도 사용하기 쉽도록 최초로 만들어진 웹 브라우저
>>> Mosaic이 1993년 최로로 등장. 그 후 넷스케이프로 대중에게 보급

- HTML5에서 웹 페이지는 어떻게 만들까?
>>> 구조와 내용은 HTML5, 출력되는 모양은 CSS3 style sheet로,
    사용자 입력을 처리하거나 웹 어플리케이션을 작성하는 것은 자바스크립트로 작성

CHAPTER 2.

- What is HTML?
>>> <태그>로 문서의 각 요소를 표현하여 웹 문서를 작성하는 언어

- <!DOCTYPE html> 이 꼭 있어야 하는가?
>>> html5 언어로 작성된 문서임을 브라우저에게 알리는 지시어로서 첫 줄에 나와야 한다.

- HTML5에서 웹페이지 파일은 기본적으로 UTF-8 코드로 저장되어야 함.

- <meta>태그는 HTML 페이지의 저작자, 페이지의 내용, 키워드 등을 표현하는데 꼭 사용해야하나?
>>> 검색 엔진은 검색 키워트를 이 태그에서 확인하기에, 웹 페이지가 잘 발견될 수 있도록 적절히 사용해야함.

-링크와 앵커의 차이점
>>> 문서내의 링크에 연결될 수 있는 특정지점을 앵커라고 부른다. 모든태그가 앵커가 될 수 있다. 앵커의 이름은 id 속성으로 작성

-웹페이지의 한 부분에 다른 웹페이지를 또 내장하는 것이 가능?
>>> <iframe> 사용하면 쌉가능!
